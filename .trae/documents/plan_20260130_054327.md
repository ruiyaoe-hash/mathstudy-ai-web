# 安装 Supabase CLI 详细计划

## 前提条件
- ✅ 已下载 `supabase_windows_amd64.tar.gz` 压缩包
- ✅ 位于 `C:\Users\13299\Downloads\` 目录
- ✅ 项目已通过 `pnpm add -D supabase` 安装了 CLI 包

## 安装步骤

### 步骤 1：创建临时解压目录
- 在 `TEMP` 目录创建 `supabase-extract` 文件夹
- 用于临时存放解压后的文件

### 步骤 2：解压 tar.gz 压缩包
- 使用 PowerShell 命令解压 `supabase_windows_amd64.tar.gz`
- 先解压 gzip 部分，再解压 tar 部分
- 提取所有文件到临时目录

### 步骤 3：定位可执行文件
- 在临时解压目录中找到 `supabase` 可执行文件
- 确认文件存在且大小合理

### 步骤 4：配置项目目录
- 在 `node_modules/supabase/` 目录创建 `bin` 文件夹
- 确保目录结构正确

### 步骤 5：复制并配置可执行文件
- 将解压出的 `supabase` 可执行文件复制到 `node_modules/supabase/bin/` 目录
- 重命名为 `supabase.exe`（Windows 系统要求）
- 验证文件权限

### 步骤 6：验证安装
- 运行 `pnpm exec supabase --version` 检查版本信息
- 确认命令能够正常执行

### 步骤 7：测试基本功能
- 运行 `pnpm exec supabase help` 测试命令响应
- 验证 CLI 基本功能正常

## 预期结果
- ✅ Supabase CLI 成功安装到项目中
- ✅ 可以通过 `pnpm exec supabase` 执行命令
- ✅ 版本信息正确显示
- ✅ 基本命令响应正常
- ✅ 为后续登录和部署 Edge Functions 做好准备
