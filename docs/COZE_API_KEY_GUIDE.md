# 🚀 Coze 平台 API Key 获取指南

## 📍 快速获取 API Key

### 情况1：通常不需要发布 ✅

大多数情况下，**创建应用后就可以直接获取 API Key**。

### 步骤：

1. **打开你的应用**
   - 在 Coze 平台首页找到你创建的应用
   - 点击进入应用编辑页面

2. **查找 API 信息**
   - 在页面右上角找到 **"..."** 三个点按钮
   - 点击它
   - 选择 **"API 信息"** 或 **"配置 API"**

3. **获取 API Key**
   - 你会看到一个 **Personal Access Token (PAT)**
   - 格式类似：`pat_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`
   - 点击复制按钮复制它

4. **完成！**
   - 把这个密钥告诉我，我帮你配置

## 🔄 如果需要发布应用

### 发布步骤：

1. 在应用编辑页面
2. 点击右上角 **"发布"** 按钮
3. 选择发布环境：
   - 🎯 选择 **"正式环境"** 或 **"生产环境"**
4. 点击 **"确认发布"**
5. 等待发布完成（通常 3-5 秒）
6. 发布完成后，再去查找 API 信息

## ✅ 如何确认 API Key 格式正确？

### 正确的格式：

```
pat_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

特点：
- ✅ 以 `pat_` 开头
- ✅ 后面是一长串字符
- ✅ 总长度大概 40-60 个字符

### 错误的格式：

❌ `app_xxx` (这是应用ID，不是API Key)
❌ `https://www.coze.com/xxx` (这是URL，不是API Key)
❌ 只有数字或只有字母 (API Key 是字母数字混合)

## 💡 常见问题

### Q: 发布应用会影响功能吗？
**A**: 不会。发布只是让应用正式上线，不影响获取 API Key。

### Q: 我找不到"API信息"选项怎么办？
**A**: 尝试先发布应用，发布后通常会显示。

### Q: API Key 会过期吗？
**A**: 一般不会过期，但建议定期更换以提高安全性。

### Q: 我可以创建多个应用吗？
**A**: 可以。但这个项目只需要一个应用的 API Key。
