# 🚀 项目下载和部署指南

由于系统限制，无法直接提供压缩包下载。请使用以下方法获取项目代码。

---

## 🎯 推荐方法：GitHub 克隆

### 快速命令

```bash
# 1. 克隆项目
git clone https://github.com/ruiyaoe-hash/demo1.git

# 2. 进入项目目录
cd demo1

# 3. 安装依赖
pnpm install

# 4. 启动开发服务器
pnpm dev
```

浏览器访问：http://localhost:5000

---

## 📦 备用方法：GitHub 网站下载

1. 访问：https://github.com/ruiyaoe-hash/demo1
2. 点击绿色 "Code" 按钮
3. 选择 "Download ZIP"
4. 解压后安装依赖

## 🔧 配置步骤

1. 创建 `.env` 文件（参考 `.env.example`）
2. 获取 Supabase API 密钥并配置
3. 获取 Coze API Key 并配置到 Supabase Secrets
4. 启动项目并测试

详细步骤请查看 `QUICK_START.md`。
